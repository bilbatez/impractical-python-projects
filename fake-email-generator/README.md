# Fake Email Generator

Generate unique emails on a single file.

```shell

|__] |___ | \| |___ |  \ |  |  |  |__| |  \     
    
Created by: Albert JT

usage: generator.py [-h] [-n NUM] [-of OUTPUT_FILENAME] [-e EXTENSION]

Fake Data Generator

optional arguments:
  -h, --help            show this help message and exit
  -n NUM, --num NUM     Number of data needs to be generated (default: 1000)
  -of OUTPUT_FILENAME, --output-filename OUTPUT_FILENAME
                        Output filename (default: 1626883345342742)
  -e EXTENSION, --extension EXTENSION
                        Output File Extension (default: .txt)
```
